# Donation Management System

A full-stack Donation Management System built using React.js and Node.js to streamline donation request handling, tracking, and role-based dashboard management.

## Features

- User Authentication (Donor/Admin)
- Role-based Dashboard
- Donation Request Management
- Status Tracking (Pending / Approved / Completed)
- Clean and Responsive UI

## Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: (Add MongoDB/MySQL if used)
- Version Control: Git & GitHub

## Installation

1. Clone the repository
2. Navigate into project folder
3. Install dependencies
4. Start the development server
